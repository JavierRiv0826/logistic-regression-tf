# Logistic Regression with TensorFlow

This project implements a simple **Logistic Regression model** using **TensorFlow**.  
It can be used as a starting point for binary classification tasks and learning purposes.

## Files
- `test1.py` — Main Python script to train and evaluate the model
- `requirements.txt` — Python dependencies
- `.gitignore` — Files/folders ignored by Git
- `README.md` — Project documentation

## Installation

**1. Clone the repository:**

```bash
git clone https://github.com/JavierRiv0826/logistic-regression-tf.git
cd logistic-regression-tf```

**2. Create and activate a virtual environment:**

Windows (PowerShell)

python -m venv venv
.\venv\Scripts\Activate.ps1

Mac/Linux

python3 -m venv venv
source venv/bin/activate

**3. Install dependencies:**

pip install -r requirements.txt

## Usage

Run the script:

python test1.py

This will train the logistic regression model and print the results.

**Notes**

Tested with TensorFlow 2.20 (see requirements.txt for exact version)

Make sure your Python version is 3.8 or above

You can modify test1.py to experiment with different datasets
